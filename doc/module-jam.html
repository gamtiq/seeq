<static> detect(name, callback)

Check whether package with the specified name is existent. Data about the found package will be passed into callback as array. If no package is found, empty array will be passed into callback.
Parameters:
Name Type Description
name String Name of the package to check.
callback function Function that should be called to process operation's result.
